I code

iOS

有近四年工作经验的iOS开发工程师。目前的行业是互联网金融。debug的能力还可以，依然有提升的空间，且找到了参考。swift的技能树只点了第一层，计划未来半年提高一到两层。

工作环境相对简单。完全没有办公室政治的工作不存在，目前在稳步发展期，但何时下一次"内乱"来临，并不知道。

虽然在互联网金融工作，本身对金融、经济、股票，毫无兴趣，唯一的兴趣可能是通过理财获得额外收益，其它方式都太费心力。

项目中遇到的最大的困难，暂时没有最大的，只会有更大的吧，而且自己会去创造困难，比如目前正在进行的MRC项目ARC化。

最有成就感的事，应该是从零开始建立了目前的团队，看着它成形、成长。

I swing

I'm not single I'm solo!

在Swing Beijing的Cat's Corner学习了半年solo课程，从level1到level3。目前跟随老师精修big apple。

目前对solo的理解是：

  • swing的基础；
  • 重要的是rhythm，不是某个step；
  • 表达自我。

Lindy Hop

作为一个follower，首先，我需要进入音乐，需要努力达到和solo一样的状态。其次，去感受leader，在连接的时候对两人的groove达成共识。

时刻保持惯性，对leader信号的反应需要形成肌肉记忆，慢慢来。

I bicycle

2016年开始，养成了较为科学的饮食习惯，增加了蔬菜在饮食结构中的比例，同时注意补充高质量的蛋白质。

通过郑多燕、keep、swing等保持了每周足够大的运动量，在此基础上，多选择骑车，增加日常锻炼。

做这些的目的只有一个，拥有一个相对健康的身体。
